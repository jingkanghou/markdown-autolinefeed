## 功能  
- 1.自动为markdown每行结尾添加换行(行尾添加两个空格)  
- 2.自动为markdown填充空格  


## 使用  
#### 换行  
- 方式一：  
快捷键 Ctrl+Alt+A  
- 方式二：  
右键菜单-->选择[Auto Linefeed]项  
- 方式三：  
Ctrl+Shift+P --> 输入 `Auto Linefeed` 并选择  

#### 填充空格  
- 方式一：  
快捷键 Ctrl+Alt+F  
- 方式二：  
右键菜单-->选择[Auto Space]项  
- 方式三：  
Ctrl+Shift+P --> 输入 `Auto Space` 并选择
